# Agent Manager

A terminal UI to manage AI coding-agent sessions on your own machine. Create and enter sessions for multiple tools (Claude Code, OpenCode, custom CLIs), organize them into groups, archive finished ones, and watch live status (working / ready / errored / idle) plus computer resource stats.

A terminal manager for tmux-hosted AI coding sessions.

Stack: Go + Bubble Tea + SQLite + tmux.

See [design spec](docs/superpowers/specs/2026-07-15-agent-manager-design.md).
